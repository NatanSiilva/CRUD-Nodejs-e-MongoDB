## 💻 Como baixar o projeto 

```bash

    # Clonar o repositório atraves GitHub Desktop
    $ git clone https://github.com/NatanSiilva/CRUD-Nodejs-e-MongoDB.git

    # Entrar no diretório
    $ cd CRUD-Nodejs-e-MongoDB

    # Instalar as dependêcias
    $ npm install 

    # Iniciar o projeto
    $  npm start
```
